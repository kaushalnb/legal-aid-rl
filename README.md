# Legal Aid RL (AI-Powered Legal Assistant)

This project is an AI-based legal assistant that helps users understand legal problems and suggests possible actions using a reasoning-based approach.

## Features
- Task-based legal assistance
- AI-powered reasoning
- Interactive UI using Gradio
- Supports multiple legal scenarios

## Tech Stack
- Python
- Gradio
- OpenAI API
- Docker

## How it Works
The system takes a legal query, analyzes it using predefined tasks and logic, and provides structured legal guidance.
